Terimakasih telah mengunjungi halaman ini. Ini adalah contoh penggunaan CSS internal untuk mengubah gaya elemen HTML secara langsung. Anda dapat melihat bahwa teks ini memiliki ukuran font yang besar dan latar belakang berwarna hitam dengan padding yang cukup besar untuk memberikan ruang di sekitarnya.

CSS internal memungkinkan Anda untuk mengelompokkan gaya dalam satu tempat, sehingga lebih mudah untuk mengelola dan memelihara kode CSS Anda. Ini sangat berguna ketika Anda ingin menerapkan gaya yang konsisten di seluruh halaman web tanpa harus menulis ulang kode CSS untuk setiap elemen.
